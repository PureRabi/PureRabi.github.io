一维随机变量

离散型随机变量

 离散型随机变量的基本定义是

  1. 对于任意的自然数$n$，有$p_n\ge0$;
  2. $\sum\limits_np_n=1$.

Bernoulli分布

 伯努利分布是最基础的离散型分布之一，如果随机变量X的分布律为：

记作$X \sim B(1,p)$。

二项分布

 如果一个随机变量X的分布律为我们就认为随机变量服从参数为$(n,p)$的二项分布，记作$X \sim B(n,p)$。由此可得，二项分布实际上就是多次独立重复的Bernoulli实验。

Poisson分布

 泊松分布在随机过程里非常重要，如果一个随机变量X的分布律为我们就认为随机变量服从参数为$\lambda$的Poisson分布，记作$X \sim \pi(\lambda)$

几何分布

 若随机变量X的分布律为我们认为随机变量X服从参数为$p$的i几何分布。可以简单的认为几何分布就是进行$k-1$次失败实验后成功了一次，反之亦然。

连续型随机变量

 连续型随机变量最基本的定义是从积分出发的，就是其中，函数$f(x)$是概率密度函数，$F(x)$是分布函数。

均匀分布

 若随机变量X的密度函数为$$
f(x) =
\begin{cases}
\frac{1}{b-a} & \text{for } a \leq x \leq b, \\
0 & \text{otherwise}.
\end{cases}

f(x; \lambda) =
\begin{cases}
\lambda e^{-\lambda x} & x \geq 0, \\
0 & x < 0.
\end{cases}

F(x; \lambda) =
\begin{cases}
1 - e^{-\lambda x} & x \geq 0, \\
0 & x < 0.
\end{cases}

f(x; \mu, \sigma) = \frac{1}{\sigma\sqrt{2\pi}} e^{-\frac{1}{2} \left(\frac{x-\mu}{\sigma}\right)^2}

f(x) = \frac{1}{\sqrt{2\pi}} e^{-\frac{x^2}{2}}

f(x; \alpha, \beta) = \frac{x^{\alpha-1}e^{-\frac{x}{\beta}}}{\beta^\alpha \Gamma(\alpha)}

\Gamma(\alpha) = \int_0^{\infty} t^{\alpha-1}e^{-t} dt

f(x; r, \lambda) = \lambda^r \frac{x^{r-1}e^{-\lambda x}}{\Gamma(r)}

 中心极限定理描述的是：当我们从一个大样本中抽取多个小样本并计算它们的均值时，这些均值的分布将趋向于正态分布，即便原始样本的分布不是正态分布。再比如说，多个正态分布的和是正态分布，多个均匀分布的和依然是正态分布。
