概率论与随机过程（一）

概率论与随机过程

一维随机变量

离散型随机变量

 离散型随机变量的基本定义是

  1. 对于任意的自然数$n$，有$p_n\ge0$;
  2. $\sum\limits_np_n=1$.

Bernoulli分布

 伯努利分布是最基础的离散型分布之一，如果随机变量X的分布律为：
$$ P(X = x) = \begin{cases}
p & \text{if } x=1, \
1-p & \text{if } x=0.
\end{cases} $$
记作$X \sim B(1,p)$。

二项分布

 如果一个随机变量X的分布律为$$P{X=k}=C_{n}^{k}p^{k}(1-p)^{n-k}$$我们就认为随机变量服从参数为$(n,p)$的二项分布，记作$X \sim B(n,p)$。由此可得，二项分布实际上就是多次独立重复的Bernoulli实验。

Poisson分布

 泊松分布在随机过程里非常重要，如果一个随机变量X的分布律为$$P{X=k} = \frac{ {\lambda^k e^{-\lambda} } }{ {k!} }$$我们就认为随机变量服从参数为$\lambda$的Poisson分布，记作$X \sim \pi(\lambda)$

几何分布

 若随机变量X的分布律为$$P{X=k}=q^{k-1}p$$我们认为随机变量X服从参数为$p$的i几何分布。可以简单的认为几何分布就是进行$k-1$次失败实验后成功了一次，反之亦然。

连续型随机变量

 连续型随机变量最基本的定义是从积分出发的，就是$$F(x)= \int_{-\infty}^{x}f(t)dt$$其中，函数$f(x)$是概率密度函数，$F(x)$是分布函数。

均匀分布

 若随机变量X的密度函数为$$
f(x) =
\begin{cases}
\frac{1}{b-a} & \text{for } a \leq x \leq b, \
0 & \text{otherwise}.
\end{cases}
$$则称随机变量X服从区间$[a,b]$上的均匀分布，记作$X\sim U[a,b]$。均匀分布的分布函数如下所示：$$F(x) =
\begin{cases}
0 & \text{for } x < a, \
\frac{x - a}{b - a} & \text{for } a \leq x < b, \
1 & \text{for } x \geq b.
\end{cases}$$易得，分布函数是由密度函数积分得来的。

指数分布

随机变量X的密度函数为$$
f(x; \lambda) =
\begin{cases}
\lambda e^{-\lambda x} & x \geq 0, \
0 & x < 0.
\end{cases}
$$并且分布函数为$$
F(x; \lambda) =
\begin{cases}
1 - e^{-\lambda x} & x \geq 0, \
0 & x < 0.
\end{cases}
$$其中，$\lambda>0$是常数，随机变量X服从参数为$\lambda$的指数分布。指数分布具有无记忆性，也就是说若已知寿命长于S年，再过T年的概率是和年龄S无关的。

正态分布

 
$$
f(x; \mu, \sigma) = \frac{1}{\sigma\sqrt{2\pi}} e^{-\frac{1}{2} \left(\frac{x-\mu}{\sigma}\right)^2}
$$当正态分布的均值为0，方差为1时，我们认为此时$X$服从标准正态分布。

$$
f(x) = \frac{1}{\sqrt{2\pi}} e^{-\frac{x^2}{2}}
$$

Gamma分布

$$
f(x; \alpha, \beta) = \frac{x^{\alpha-1}e^{-\frac{x}{\beta}}}{\beta^\alpha \Gamma(\alpha)}
$$伽马函数定义为：

$$
\Gamma(\alpha) = \int_0^{\infty} t^{\alpha-1}e^{-t} dt
$$由于不同的学者有不同的习惯，这里可以设置参数$\lambda$和$r$, 且$r=\alpha$，$\lambda = \frac{1}{\beta}$，由此我们可以得到$$
f(x; r, \lambda) = \lambda^r \frac{x^{r-1}e^{-\lambda x}}{\Gamma(r)}
$$也就是教材上的公式。
 当$r=1$时，Gamma函数为1，此时Gamma分布退化为指数分布，那么说明指数分布是Gamma分布的一个特例。
 概率论中最重要的定理是大数定理和中心极限定理。中心极限定理又是最最重要的定理。
大数定理说的是样本数量越多，则其平均就越接近期望值。也就是
实验的次数越多，实验的结果越接近理论值。用数学表达式也很简单：
$$\lim\limits_{n\rightarrow\infty}P(|\frac{S_n}{n}-p|<\epsilon)=1$$
 中心极限定理描述的是：当我们从一个大样本中抽取多个小样本并计算它们的均值时，这些均值的分布将趋向于正态分布，即便原始样本的分布不是正态分布。再比如说，多个正态分布的和是正态分布，多个均匀分布的和依然是正态分布。

随机变量的数字特征

数学期望

期望定义

 首先看看数学期望，数学期望也就是均值，对于离散型和连续型两种随机变量有不同的期望计算方法。
 期望代表了随机变量的加权平均，用于衡量随机变量的平均值（中心值）：
$$E(X)=\sum\limits_{i}\limits^\infty x_{i}p(x_{i})$$
$$E(X)=\int_{-\infty}^{\infty}xf(x)dx$$
如果说对于两个独立的分布，那么其乘积XY的期望如下：
$$E(X)=\sum\limits_{i}\limits^\infty\sum\limits_{j}\limits^\infty x_{i}y_{j}p(x_{i},y_{j})$$
$$E(X)=\int_{-\infty}^{\infty}\int_{-\infty}^{\infty}xyf(x,y)dydx$$不过，由于其独立性，后续会说到更简单的计算公式。

期望性质

 数学期望也有诸多性质：

  1. $E[c]=c$ 若$a\le X \le b$，那么$a\le E(X) \le b$
  2. $E[cX]=cE[X]$
  3. $E[aX+bY]=aE[X]+bE[Y]$，$E(\sum\limits_{i=1}^na_{i}X_{i})=\sum\limits_{i=1}^na_{i}E[X_{i}]$
  4. 若X，Y独立，那么$E[XY]=E[X]E[Y]$

方差

方差定义

 接下来看看方差。方差描述的是独立变量偏离其期望值的程度。其计算公式如下：
$$V(X)=E[X^2]-E^2[X]$$

方差性质

 方差也有诸多性质：

  1. $D(X)\ge 0$ 若$c$是常数，那么$D(c)=0$
  2. $D(cX+d)=c^2D(X)$
  3. $D(aX+bY)=a^2D(X)+b^2D(Y)+2abE(E-E[X])(Y-E[Y])$
  4. $D(X)=0\Leftrightarrow P{X=c}=1,c=E[X]$

Chebyshev不等式

 切比雪夫不等式提供了一个无需知道精确分布的情况下，评估随机变量的值偏离其期望值的程度。对于任何具有有限期望值 ( \mu ) 和有限非零方差 ( \sigma^2 ) 的随机变量 ( X )，以及任意的 ( k > 0 )，切比雪夫不等式断言：
$$
P(|X - \mu| \geq k\sigma) \leq \frac{1}{k^2}
$$这意味着随机变量 ( X ) 的值落在距离其期望值 ( \mu ) 多于 ( k ) 倍标准差 ( \sigma ) 的概率不会超过 ( \frac{1}{k^2} )。

协方差

 协方差衡量两个随机变量之间的线性关系强度和方向。
$$Cov(X,Y)=E(XY)-E[X]E[Y]$$

相关系数

 相关系数是协方差除以两个随机变量的标准差的乘积，用于衡量两个随机变量之间的线性相关程度。它是协方差和标准差的比值。
$$\rho(X, Y) = \frac{Cov(X, Y)}{\sigma_X \sigma_Y}\in [-1,1]$$
从这个公式我们可以看出，如果想要得到两个随机变量的相关性，相关系数是最好的参数。为什么呢？因为在协方差中，协方差很容易受到单个随机变量自己的离散程度的影响，而当两者都除以标准差后，这种差异就被去除了。

重要的数字特征

二项分布
  • 期望值: ( np )
  • 方差: ( np(1-p) )
泊松分布
  • 期望值: ( \lambda )
  • 方差: ( \lambda )
均匀分布
  • 期望值: ( \frac{a + b}{2} )
  • 方差: ( \frac{(b - a)^2}{12} )
指数分布
  • 期望值: ( \frac{1}{\lambda} )
  • 方差: ( \frac{1}{\lambda^2} )
均匀分布
  • 期望值: ( \frac{a + b}{2} )
  • 方差: ( \frac{(b - a)^2}{12} )
正态分布
  • 期望值: ( \mu )
  • 方差: ( \sigma^2 )
Gamma分布
  • 期望值: ( \alpha\beta )
  • 方差: ( \alpha\beta^2 )

随机变量的关系

 很好，有了这些数学特征作为指标，我们接下去来看看随机变量的之间的关系。随机变量之间的关系有：独立、相关和正交。

随机变量的独立性

 对于两个随机变量X和Y，我们通常这样来判断其独立性，如果，对于连续型随机变量：
$$f(x,y)=f(x)f(y)$$
对于离散型随机变量：
$$P(XY)=P(X)P(Y)$$
 那么我们认为，X和Y是独立的（顺带一提，有监督模型的大前提是，独立同分布）。两个变量独立，意味着其取值互不影响，即一个随机变量的取值不提供关于另一个随机变量的任何信息。简而言之，两个随机变量X和Y相互独立意味着它们的联合分布等于各自的边缘分布的乘积，那么，我们在之前所做的XY之前的期望就有了更简单的做法：
$$E[XY]=E[X]E[Y]$$

随机变量的相关性

接下来看看相关性。相关性一般用相关系数来进行判断。如果（显然）：
$$\rho(X, Y)=0$$则两个随机变量是不相关的，反之如果相关系数为正，则两个随机变量为正相关，如果相关系数为负，两个随机变量为负相关。

随机变量的正交

 最后我们来看看什么叫随机变量之间的正交。随机变量的正交听起来很奇怪，广义上说，如果两个变量的内积是0，那么这两个变量就是正交的。同样的：
$$E[XY]=E[X]E[Y]=0$$
 那么我们称X和Y是正交的，也就是联合期望为0,协方差也为0。所以，想要得到X和Y正交，必须是如果$X>0$，$Y$需要有正有负，反之亦然。
 接下来看一些有意思是事情：既然$\rho(X, Y)=0$，那么也就是$Cov(X,Y)=0$，也就是：
$$E[XY]=E[X]E[Y]$$
也就是说，如果两个随机变量是不相关的，那么就会得到独立的性质而不是推出独立！，
相反，如果两个随机变量是独立的，那么$E[XY]-E[X]E[Y]=0$，也就是$Cov(X,Y)=0$，推出了不相关。
$$\textbf{独立}\rightarrow\textbf{不相关}$$
 正交是最复杂的性质，可以这么思考：如果两个随机变量独立或不相关（因为独立和不相关具有相似性质，$Cov(X,Y)=0$），且$E[X]=0\quad or\quad E[Y]=0$时，我们认为X和Y是正交的。如果X和Y是正交的，且$E[X]=0\quad or\quad E[Y]=0$，那么X和Y是不相关的。
 总结一下就是：
$$\textbf{独立}\rightarrow\textbf{不相关}\xleftrightarrow{E[X]=0 or E[Y]=0}\textbf{正交}$$

多维随机变量

二维离散型随机向量

定义

 若存在二维随机向量(\boldsymbol{X},\boldsymbol{Y})的取值是有限个或可列无穷个数对，则称(\boldsymbol{X},\boldsymbol{Y})为二维离散型随机变量。
$$p_{ij}=P{X=x_i,Y=y_i}$$为二维离散型随机变量的联合分布律。

二维离散型随机向量的边缘分布函数

变量(\boldsymbol{X})关于二维随机变量((\boldsymbol{X},\boldsymbol{Y}))的边缘分布。

$$p_{i.}=\sum\limits_jp_{ij}$$反之同理。

连续型随机向量

定义

$$F(x_1,\dots,x_n)=\int_{-\infty}^{x_1}\dots\int_{-\infty}^{x_n}f(u_1,\dots,u_n)du_1\dots d_n$$
显然，如果是二维的连续型随机向量，分布函数的物理含义是一块矩阵面积。
 如果要求(X_k)边缘密度函数，则需要将剩下的(k-1)个密度函数进行积分即可。

二维正态分布

 二维正态分布的边缘分布是一维正态分布，且其分布与二维正态分布的参数r无关。
